## 一、项目背景

### 痛点

- 人工编写实施文档耗时长，一个变更可能需要数小时
- 手动填写变更申请流程繁琐，重复劳动多
- 文档格式不统一，审核沟通成本高

### 解决方案

**OpsPilot** - Excel 清单到 Word 实施文档的自动化转换工具

**改变原先人工提变更的过程**：从 Excel 清单一键生成规范化的 Word 实施方案

### 核心价值

| 价值 | 说明 |
|------|------|
| 提升效率 | 自动化解析与生成，节省 80% 文档编写时间 |
| 规范输出 | 统一文档格式，符合企业标准 |
| 流程变革 | 改变人工提变更的传统方式，实现文档自动化 |

## 二、核心功能

### 1. 智能解析

- 支持多 Sheet 结构的 Excel 文件
- 自动提取任务名、操作类型、部署单元等核心字段
- 动态表头解析，灵活适配不同格式

### 2. 模板渲染

- 基于 docxtpl + Jinja2 模板驱动
- 确保输出样式与样板完全一致
- 模板失败自动回退内置渲染

### 3. 章节排序

- 根据 rules.yaml 自动排序 Word 章节顺序
- 空数据章节自动跳过
- 支持可视化拖拽排序

### 4. 任务聚合

- 相同操作类型的任务自动合并处理
- 生成规范化表格
- 减少文档冗余

### 5. 风险识别

- 自动扫描删除、下线、重建等高危操作
- 触发挂起确认流程
- 防止误操作导致事故

### 6. MCP 服务化

- 通过 FastMCP 对外暴露标准化接口
- 支持 Cursor 等 IDE 集成
- 实现 Agent 协同工作

### 7. Web 配置中心

- 可视化界面管理规则配置
- 支持拖拽排序、富文本编辑
- 版本回滚、YAML 预览

## 四、数据流程

```
┌──────────┐     ┌──────────┐     ┌──────────────┐
│  Excel   │ ──▶ │  Parser  │ ──▶ │ report.json  │
│  清单    │     │  解析器   │     │  中间数据    │
└──────────┘     └──────────┘     └──────────────┘
                                        │
                                        ▼
┌──────────┐     ┌──────────┐     ┌──────────────┐
│  Word    │ ◀── │ docxtpl  │ ◀── │ report.json  │
│  实施文档 │     │  渲染器   │     │              │
└──────────┘     └──────────┘     └──────────────┘
```

### 执行流程

1. **解析阶段**：读取 Excel 清单，提取结构化数据
2. **渲染阶段**：基于模板生成 Word 实施文档

|------|----------|---------------|
| 准备清单 | Excel 手动填写 | Excel 手动填写 |
| 编写文档 | 数小时手动编写 | **一键自动生成** |

### 典型使用场景

**场景：Agent 对话式变更提单**

```
用户: 我这周有一个变更，这是我的上线checklist，请你帮我检查下然后自动提个变更单
      [上传 Excel 文件]

Agent: 好的，我来帮你处理...
       ✓ 已解析 Excel 清单，共 3 个 Sheet，12 项操作
       ✓ 已生成 Word 实施方案
       [返回 Word 文档]

      请确认实施方案是否正确？

用户: [查看 Word] 确认无误，提交吧

Agent: 正在提交变更单...
       ✓ 已生成在线变更单
       [返回变更单链接]

      请点击链接审核变更单详情，确认后可提交

用户: [点击链接] → [平台确认提交] ✓
```

### 交互流程详解

| 步骤 | 角色 | 动作 | 技术实现 |
|------|------|------|----------|
| 1 | 用户 | 发起对话 + 上传 Excel | Agent 对话界面 |
| 2 | Agent | 调用 MCP 解析 Excel | OpsPilot MCP 服务 |
| 3 | Agent | 生成 Word 文档 | OpsPilot MCP 服务 |
| 4 | 用户 | 复核 Word 确认 | - |
| 5 | Agent | 调用平台 AI 提单 | 平台 API |
| 6 | 平台 | 生成在线变更单 | 平台服务 |
| 7 | 用户 | 平台确认提交 | 变更管理平台 |

**效率提升**：从数小时的人工流程 → 分钟级 Agent 对话完成
